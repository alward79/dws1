Definitions

Headings
Headings and sub-headings reflect the hierarchy of information.
<h1> A main heading.
<h2> A level 2 heading.
<h3> A level 3 heading.
<h4> A level 4 heading.
<h5> A level 5 heading.
<h6> A level 6 heading.
<p>
Paragraph - Displays text on the HTML page.
<b>
Bold - Change words to bold font.
<i>
Italic - Change words to italic font.
<sup>
Superscript - Appear above the baseline of the text. exp: 102
<sub>
Subscript - Appear below the baseline of the text. exp: CO2
<br/>
Line Breaks - Add a line break inside the middle of a paragraph.
<hr/>
Horizontal Rule - Add a horizontal rule between sections
<strong>
Strong - Shows that the element has a strong importance
<em>
Emphasis - Indicates emphasis that subtly changes the meaning of a sentence. By default, browsers will show the contents of an <em> element in italic.
<blockquote>
Blockquote - Is used for long quotes that usually take up an paragraph.
<q>
Quote - Used to add quotation to a sentence.
<abbr title = ""> and <acronym title = "">
Abbreviation & Acronym - Creates the abbriviation or acronym but displays the meaning of it when hovered over the element.
<cite>
Citation - Creates a citation to reference outside material.
<dfn>
Definitions - Used when defining a new term for the first time.
<address>
Address - Contains contact information for the author of the page.
<ins>
Insert - Can be used to show content that has been inserted into a document, and its usually underlined.
<del>
Delete - Shows text that has been deleted, and usually has a line through it.
<s>
Strikethrough - Used when something is no longer accurate.
<ol>
Ordered list - Used when each item in a list is numbered.
<ul>
Unordered List - List that begins with bullet points.
<li>
List Item - Each item in the list is placed between an opening a closing <li> tag.
<dl>
Definition List - Usually consists of a series of terms and their definitions.
<dt>
Definition Term - Used to contain the term being defined.
<dd>
Definition - Used to contain the definition for the term.
