Definitions

Headings
Headings and sub-headings reflect the hierarchy of information.
<h1> A main heading.
<h2> A level 2 heading.
<h3> A level 3 heading.
<h4> A level 4 heading.
<h5> A level 5 heading.
<h6> A level 6 heading.
Paragraph
<p> Displays text on the HTML page.
Bold
<b> Change words to bold font.
Italic
<i> Change words to italic font.
Superscript
<sup> Appear above the baseline of the text. exp: 102
Subscript
<sub>. Appear below the baseline of the text. exp: CO2
Line Breaks
<br/> Add a line break inside the middle of a paragraph.
Horizontal Rule
<hr/> Add a horizontal rule between sections
Strong
<strong> Shows that the element has a strong importance
Emphasis
<em> Indicates emphasis that subtly changes the meaning of a sentence. By default, browsers will show the contents of an <em> element in italic.
Blockquote
<blockquote>. Is used for long quotes that usually take up an paragraph.
Quote
<q> Used to add quotation to a sentence.
Abbreviation & Acronym
<abbr title = ""> and <acronym title = ""> Creates the abbriviation or acronym but displays the meaning of it when hovered over the element.
Citation
<cite> Creates a citation to reference outside material.
Definitions
<dfn> Used when defining a new term for the first time.
Address
<address>. Contains contact information for the author of the page.
Insert
<ins> Element can be used to show content that has been inserted into a document, and its usually underlined.
Delete
<del> Element can show text that has been deleted from it, and usually has a line through it.
Strikethrough
<s> Used when something is no longer accurate.
Ordered list
<ol> Used when each item in a list is numbered.
Unordered List
<ul> List that begins with bullet points.
List Item
<li> Each item in the list is placed between an opening a closing <li> tag.
Definition List
<dl> Usually consists of a series of terms and their definitions.
Definition Term
<dt> Used to contain the term being defined.
Defintion
<dd> Used to contain the definition for the term.
